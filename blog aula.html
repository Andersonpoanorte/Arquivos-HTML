Visual Studio Code

Nesta aula entendemos que o Visual Studio Code, é uma ferramenta open source, sendo gratuita, essa ferramento está diponivél para Mac, Linux e Windows.

O Visual Studio Code, ajuda você a codificar rapidamente. Pode ser usado para codificar em qualquer linguagem de progrmação, sem trocar de editor, além de ter suporte para varias linguagens. Para fazer download da ferramenta basta entrar no site https://code.visualstudio.com/.

Inspetor de Elemento

Nesta aula aprendi sobre a ferramenta Inspetor de elemento, onde é um utilitário que permiti visualizar o código fonte subjacente de qualquer página web. Além disso você pode usa-lá para fazer alterações temporárias e ver os resultados em tempo real, deixando o código-fonte original intacto.

Isso é extremamente útil se você precisa testar uma alteração ou diagnosticar um problema. Também pode ser de grande valia se você encontrar um site com um recurso de que goste e estiver curioso de saber como ele foi implementado.

A maioria dos principais navegadores, incluindo Ghrome, Firefox e Safari, oferece algumas variações dessa ferramentas.

Estrutura básica html

Nesta aula aprendi sobre estrutura básica de html, onde consiste iniciando com html, head, body, tudo que estiver em body vai aparecer para o usuário, o head vai carregar antes de aparecer para o usuário, temos em head a tag title, que é o titulo da página.

A estrutura é feita por varias tags e atribuições, entre outras funções, vejamos abaixo uma estrutura básica de html:

< ! DOCTYPE html >
< html lang="en" >
< head >
< meta charset="UTF-8" >
< meta http-equiv="X-UA-Compatible" content="IE=edge" >
< meta name="viewport" content="width=device-width, initial-scale=1.0" >
< title>Document< /title >
< /head >
< body >
< /body >
< /html >

Então com essas funções podemos iniciar a estrutura de um site, sendo isso a parte básica.

Tags html

Aprendi nessa Aula sobre tags html, as tags tem a função de estruturar a página web, pois Cada página de um site é composta por tags de HTML. Elas são códigos que orientam a estrutura do documento, como o tamanho, a fonte e as quebras de linhas.

Com essas marcações, o código de HTML cria estruturas e hierarquização das informações, dividindo-as em seções, o que ajuda na jornada do cliente.

Segue algumas tags que aprendi a utilizar:

  • head: define o início do HTML;
  • title: indica o título;
  • body: é usado para demonstrar o início do conteúdo;
  • h1,h2,h3: tags utilizadas para indicar títulos e subtítulos;
  • strong: acrescenta o negrito ao texto;
  • i:sinaliza a formatação do conteúdo em itálico;
  • p: insere o parágrafo
  • a href=LINK>: tag usada para inserir um link.

Atributos das Tags

Aprendi nessa aula sobre os atributos das tags de html, os atributos são propriedades que uma tag pode ter, os atributos html são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento.

Os atributos HTML são um modificador de um tipo de elemento HTML. Com eles podemos identificar melhor um elemento, informar qual arquivo aquela tag deve utilizar, indicar o tipo de um campo de texto, etc

Alguns atributos que aprendi, foram, input, img, type text ou number, width.

Listas Ordenadas e Não Ordenadas

Aprendi na aula sobre listas ordenadas e não ordenadas, essas são as principais do html, a principal diferença entre a lista ordenada e não ordenada, está na maneira com que os itens de cada lista são aprensentados: a lista ordenada enumera os elementos; a lista não ordenada coloca marcadores antes de cada elemento

As listas não numeradas são usadas para listar itens, sem se preocupar com sua sequência. Chamamos de lista de marcadores apenas.

As tags usadas para criar uma lista não ordenada são ul e il. A tag "ul" é usada para definir a lista e a tag "li" é usada para cada item da lista.

As listas ordenadas ou numeradas são usadas para indicar alguma sequência ou numeração.

As tags usadas para criar uma lista ordenada são ol e li. A tag "ol" é usada para definir a lista e a tag "li" é usada para cada item da lista.

Links

Aprendi nessa aula sobre como utilizar os links no html onde vai linkando uma pagina na outra, utilizado uma ancor sendo o a href.

O elemento html link, especifica as relações entre o documento atual e um recurso externo. Possíveis usos para este elemento incluem aa definição de uma estrutura relacional para navegação. Este elemento é mais usado para vincular as folhas de estilos
